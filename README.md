# Birko.Data.Sync.Sql

SQL sync knowledge item implementation for the Birko Framework. Maps to `SyncKnowledge` table.

## Installation

```bash
dotnet add package Birko.Data.Sync.Sql
```

## Dependencies

- Birko.Data.Core (AbstractModel)
- Birko.Data.Stores (store interfaces, Settings)
- Birko.Data.Sync
- Birko.Data.SQL
